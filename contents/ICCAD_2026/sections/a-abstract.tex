\begin{abstract}
Resistive random-access memories (ReRAMs) exhibit stochastic switching dynamics that make them attractive entropy sources for hardware true random number generators (TRNGs). Prior work has exploited the non-deterministic write latencies of commercial-off-the-shelf (COTS) ReRAM devices for this purpose, but relies on inefficient von Neumann extraction to suppress structural biases. Recent work has attributed observed structural biases to error correcting codes (ECCs) embedded within COTS ReRAM memories. These ECCs inject timing perturbations from additional switching events, and are fully recoverable via side-channel analysis. In this work, we apply knowledge of recovered ECCs to directly de-bias COTS ReRAM entropy sources, reducing the need for aggressive conditioning. We evaluate our approach using the NIST SP 800-90B test suite and demonstrate a TRNG that passes NIST SP 800-22 and PractRand while achieving 17x extraction bit rate performance improvement over prior methods.
